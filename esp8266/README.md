# 感知节点说明

## 传感器说明

使用到的传感器如下所示:

+ DHT11温湿度传感器
+ 光照传感器
+ 烟雾传感器

### DHT11传感器


| 传感器引脚   | 连接单片机引脚  |
| ------------ | --------------- |
| GND          | GND             |
| VCC          | VCC             |
| 数据引脚     | D3              |

### 光照传感器


| 传感器引脚   | 连接单片机引脚  |
| ------------ | --------------- |
| GND          | GND             |
| VCC          | VCC             |
| 数据引脚     | D7              |

### 烟雾传感器


| 传感器引脚   | 连接单片机引脚  |
| ------------ | --------------- |
| GND          | GND             |
| VCC          | VCC             |
| 数据引脚     | A0              |
